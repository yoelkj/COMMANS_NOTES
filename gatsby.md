#Set Up Your Development Environment

INSTALL CLI:
```sh
npm install -g gatsby-cli
```

SHOW VERSION:
```sh
gatsby --version
```

SHOW HELP:
```sh
gatsby --help
```

CREATE PROJECT FROM GITHUB HELLO WORLD:
```sh
gatsby new gatsby_tutorial https://github.com/gatsbyjs/gatsby-starter-hello-world
```

RUN DEVELOP MODE:
```sh
gatsby develop
```
